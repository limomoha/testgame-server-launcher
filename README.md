# testgame-minecraftknockoff

This is a simple game I thought of where you mine around in a top down system with people, hit them, carry them, give them, and build with them.

To run the game, firstly, you must run testgameserver.py first or it won't work, you don't need to if another person ran it already.
After you're done —— Run testgamelauncher.py to launch the game, enter the private ip if its the local network, public ip if its over the internet, and blank if you're the host, and then enter you're name.
I understand this game is easily cheatable, but I will assume you can manage that
This is mostly assisted by AI if you don't mind.


To use the server, you don't need to download anything. To use the launcher, you need to do:

```batch/bash/zsh
pip install pygame
```

Then double click it or enter the file name in the terminal, or python3 /path/to/file.ext
Instructions for gameplay:

WASD to walk
Left and right click to mine and place.
Dirt is strong because earth IRL is strong.
Hold left click to mine.
Eat food with E.
Give something with R.
Change inventory with 1, 2, 3 and 4.
You have infinite inventory.
Diamond is rarest, Gold is less rare, food is less rare, and dirt is most common.

# Server
To set up server, simply run it, and you should see it give you the IP, and you will see everything, and all log will be saved in a file called log.txt.
To get the logs, use notepad.exe or an application to open or, it just double click.
